# Ram Parmar Shopify Portfolio

Single-page Next.js 15 portfolio for Ram Parmar, a Shopify Developer and Shopify App Developer.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn-style UI primitives
- Lucide Icons
- next-themes
